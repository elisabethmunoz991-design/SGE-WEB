¿Qué es Dolibarr?

Dolibarr es un ERP/CRM gratuito y de código abierto pensado para pequeñas y medianas empresas, autónomos y asociaciones. Funciona desde el navegador sin necesidad de instalar nada en cada equipo, y puedes activar solo los módulos que necesitas: si no usas un módulo, simplemente no aparece.

Es una de las opciones más sencillas del mercado para empezar con un ERP sin complicaciones ni costes de licencia.

Instalación

Para instalar Dolibarr en Windows usamos DoliWamp, un paquete que incluye todo lo necesario: Apache, MySQL, PHP y el propio Dolibarr. Sin complicaciones de configuración manual.

  1. Descargar DoliWamp desde la web oficial de Dolibarr.
  2. Ejecutar el instalador y seguir el asistente (siguiente, siguiente, instalar).
  3. Al terminar, DoliWamp aparece en la bandeja del sistema.
  4. Abrir el navegador y acceder a http://localhost.
  5. Seguir el asistente de configuración inicial de Dolibarr.
  6. Crear el usuario administrador y acceder al panel principal.

Proceso de instalación

Administración y configuración

Desde el menú Configuración el administrador puede ajustar todo lo relacionado con el funcionamiento del sistema. Las opciones más importantes son:

  • Módulos: activar o desactivar funcionalidades según lo que necesite la empresa.
  • Empresa: nombre, logo, moneda e idioma.
  • Usuarios: crear cuentas y asignar roles y permisos.
  • Parámetros: numeración de facturas, IVA, formatos de fecha, etc.

Administración y configuración

Acceso seguro: Roles y privilegios

Dolibarr tiene un sistema de permisos bastante completo que permite controlar exactamente qué puede hacer cada usuario dentro del sistema. Se gestiona desde Configuración → Usuarios y Grupos.

  • Administrador: acceso total a todas las funciones y configuraciones del sistema.
  • Gestor: puede crear y editar documentos pero no acceder a la configuración general.
  • Usuario básico: acceso limitado a módulos concretos o solo lectura.
  • Los permisos se asignan módulo a módulo: facturas, clientes, almacén, contabilidad, etc.

Roles y permisos

Elaboración de informes

Dolibarr permite generar informes desde los distintos módulos activos. Desde facturación puedes sacar listados de facturas, y desde el panel principal tienes acceso a estadísticas generales del sistema.

  • Exportación a PDF de facturas, presupuestos y albaranes directamente desde cada documento.
  • Exportación de listados a CSV para trabajar con los datos en Excel u otras herramientas.
  • Estadísticas y gráficos disponibles desde el panel principal del sistema.
